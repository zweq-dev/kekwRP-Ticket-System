## Kullanım/Örnekler

*/src/settings/Bot.json* dosyasından botun ayarlarını yapın,
*/src/settings/Ticket.json* dosyasındaki isim ve idleri girin.

## Yükleme 

Modülleri **npm** kullanarak yükleyin

```bash 
  npm install
  node main.js
```
